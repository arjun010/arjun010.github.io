Mixed-Initiative Tools for Data Science

Abstract

In this series of projects, I explored the design of intelligent systems that draw inferences from user interactions and complement them with automated recommendations. In doing so, these systems provide real-time guidance to users during common data science tasks including data transformation (converting data from one format into another), exploratory data analysis, data-driven communication, and interpreting machine learning models.
